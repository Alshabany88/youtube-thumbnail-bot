## ✨ المميزات

✅ تحميل صور الفيديوهات بأعلى دقة متاحة (حتى 1080p)
✅ دعم جميع صيغ الروابط (youtube.com, youtu.be, shorts)
✅ عرض عنوان الفيديو مع الصورة
✅ معالجة عدة روابط في رسالة واحدة
✅ إظهار نسبة التقدم أثناء التحميل
✅ نظام احتياطي (Fallback) إذا فشل API
